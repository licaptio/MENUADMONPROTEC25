Flujo:
1. Pantalla de carga.
2. Pantalla principal limpia.
3. Buscar cliente por ID, nombre, RFC, teléfono o email.
4. Seleccionar cliente para ver ficha completa.
5. Editar desde modal.
6. Alta cliente nuevo desde botón superior.
7. Desactivar cliente en lugar de borrarlo físicamente.
