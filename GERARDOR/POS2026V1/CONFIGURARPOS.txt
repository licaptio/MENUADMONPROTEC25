PROVSOFT POS - IMPRESION WINDOWS

REQUISITOS

1. Instalar driver Epson APD 6.12
2. Configurar impresora Epson TM-T20III
3. Establecer impresora como predeterminada
4. Desactivar:
   "Dejar que Windows administre mi impresora predeterminada"

5. Ejecutar Chrome con:

   --kiosk-printing

Ejemplo:

"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing
